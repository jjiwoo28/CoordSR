# Neural Light Field Super Resolution (NeuLF-SR)

## 프로젝트 개요
본 프로젝트는 저해상도 라이트필드 이미지를 고해상도로 변환하는 딥러닝 기반 초해상화 모델을 구현합니다.

## 주요 기능
- 라이트필드 이미지의 해상도를 2배, 4배, 8배로 향상
- Coordinate-based MLP와 CNN을 결합한 하이브리드 아키텍처 
- 다양한 스케일에서의 초해상화 성능 검증

## 설치 방법

### 프로젝트 구조
CoordSR(Coordinate-based Super Resolution)는 좌표 기반 신경망을 활용한 초해상도 모델입니다. 이 프로젝트는 다양한 모델(CoordX, R2L)을 학습하고 평가하는 스크립트들을 포함하고 있습니다.
